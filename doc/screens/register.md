## イベント処理概要

### No.4 登録ボタン押下

**INPUT**
<div class="io-item">

| 項目名 | 備考 |
| :--- | :--- |
| ユーザー名 | 項目No.1の値 |
| メールアドレス | 項目No.2の値 |
| パスワード | 項目No.3の値 |

</div>

**処理**
1. 新規ユーザーを登録する
   - バックエンド「ユーザー作成API」をコールする
     リクエストパラメータ
     　username = ユーザー名
     　email = メールアドレス
     　password = パスワード
   →画面表示：登録成功時、ログイン画面へ遷移。失敗時、エラーメッセージを表示（処理終了）
