# Analysis Methods

ユーザー独自のスクリーニング条件 (分析手法) を作成・管理し、実行結果を確認します。

## 画面イメージ
![Analysis Methods Screen](../images/analysis_methods.png)

## 役割
ユーザー独自のスクリーニング条件 (分析手法) を作成・管理し、実行結果を確認する。

## 主要要素
*   **新規手法の作成フォーム** (左カラム):
    *   手法名、説明
    *   **スクリーニング条件 (AND条件)**: `指標 (セレクト)`, `比較演算子 (>=, <=, == など)`, `数値` の動的リスト
    *   条件追加・削除ボタン、保存ボタン
*   **保存済み手法リスト** (右カラム上部):
    *   手法名、説明、設定されている条件バッジのリスト
    *   「実行 (Screen)」ボタン、「削除」ボタン
*   **スクリーニング結果** (右カラム下部 / 実行時のみ表示):
    *   表形式: `証券コード`, `企業名`, `最新書類の発行日`, `合致した指標の実績値`

#### 利用可能な指標一覧 (Available Metrics)
スクリーニング条件として利用可能な財務指標の一覧です。

| 指標名 (日本語) | 識別子 (ID) | 単位 | 説明 |
| :--- | :--- | :--- | :--- |
| **売上高** | `net_sales` | 円 | 企業の主たる営業活動から得られた収益 |
| **営業利益** | `operating_income` | 円 | 売上高から売上原価および販管費を差し引いた利益 |
| **経常利益** | `ordinary_income` | 円 | 営業利益に営業外収益・費用を加減した利益 |
| **当期純利益** | `net_income` | 円 | 税金等を差し引いた最終的な利益 |
| **総資産** | `total_assets` | 円 | 企業が保有するすべての資産の合計 |
| **純資産** | `net_assets` | 円 | 総資産から負債を差し引いた自己資本 |
| **営業CF** | `operating_cf` | 円 | 本業による現金収支 |
| **投資CF** | `investing_cf` | 円 | 設備投資や資産売却による現金収支 |
| **財務CF** | `financing_cf` | 円 | 資金調達や返済による現金収支 |
| **1株当たり純利益 (EPS)** | `eps` | 円 | 当期純利益を、発行済株式数で割った値 |
| **1株当たり純資産 (BPS)** | `bps` | 円 | 純資産を、発行済株式数で割った値 |
| **ROE (自己資本利益率)** | `roe` | % | 自己資本を用いてどれだけ効率的に利益を上げたか |
| **ROA (総資産利益率)** | `roa` | % | 総資産を用いてどれだけ効率的に利益を上げたか |
| **自己資本比率** | `equity_ratio` | % | 総資産に対する純資産（自己資本）の割合 |
| **営業利益率** | `operating_margin` | % | 売上高に対する営業利益の割合 |

## モーダル
*   **削除確認モーダル**: 手法の削除実行前の確認。
