## イベント処理概要

### No.3 ログインボタン押下

**INPUT**
<div class="io-item">

| 項目名 | 備考 |
| :--- | :--- |
| メールアドレス | 項目No.1の値 |
| パスワード | 項目No.2の値 |

</div>

**処理**
1. ユーザー認証を実行する
   - バックエンド「ログインAPI」をコールする
     リクエストパラメータ
     　username = メールアドレス
     　password = パスワード
   →画面表示：認証成功時、ダッシュボードへ遷移。失敗時、エラーメッセージを表示（処理終了）
