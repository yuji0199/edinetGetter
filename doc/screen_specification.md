# 画面設計書 (Screen Specifications)

本ドキュメントは、EDINETデータ分析アプリケーションのフロントエンド画面における設計・仕様をまとめたものです。

## 概要

本アプリケーションは、React と React Router を用いて構築されたSPA (Single Page Application) です。ナビゲーションバーを共通コンポーネントとして持ち、各機能はページコンポーネントとして分割されています。

---

## 1. 共通レイアウト

### 1.1 ナビゲーションバー (Navbar)
*   **役割**: アプリケーション全体のグローバルナビゲーション。
*   **配置**: 画面上部 (ヘッダー部分)
*   **要素**:
    *   ロゴ / アプリケーション名 (ダッシュボードへのリンク)
    *   リンクメニュー: ダッシュボード、分析手法、ポートフォリオ
    *   ユーザー情報: ログインユーザー名、ログアウトボタン

---

## 2. 画面詳細

### 2.1 ダッシュボード画面 (Dashboard)
*   **パス**: `/dashboard`
*   **役割**: ログイン直後に表示されるホーム画面。主要機能へのショートカットと、最近のデータ更新状況を提供する。
*   **主要要素**:
    *   **ヘッダーメッセージ**: 「おかえりなさい、〇〇さん」
    *   **クイックアクセスカード** (3列グリッド):
        1. 銘柄検索 (アイコンと説明)
        2. 分析手法 (アイコンと説明)
        3. ポートフォリオ (アイコンと説明)
    *   **最近同期された書類**:
        *   表形式: `企業 (証券コードとリンク)`, `書類ID`, `抽出された指標数`を表示
        *   データが空の場合はプレースホルダーを表示
    *   **EDINETデータ同期フォーム**:
        *   対象日入力 (`input type="date"`)
        *   「今すぐ同期」ボタン
        *   処理結果メッセージ (成功 / エラー) 表示領域

### 2.2 銘柄詳細画面 (StockDetail)
*   **パス**: `/stocks/:code`
*   **役割**: 特定の企業（銘柄）の詳細なハイライトと財務データを視覚的に表示する。
*   **主要要素**:
    *   「ダッシュボードに戻る」リンク
    *   **ヘッダーセクション**:
        *   企業名、証券コード、業種ラベル
        *   現在株価、PER (実績)、PBR (実績)
        *   **「ポートフォリオに追加」ボタン** (モーダルをトリガー)
    *   **データ可視化セクション** (書類データが存在する場合):
        *   **売上高と収益性グラフ**: BarChart (Recharts) を用いた棒グラフ (売上高、営業利益、純利益)
        *   **最新の財務指標詳細**: EPS, BPS, ROE, ROA, キャッシュフローなどをカード形式のグリッドで表示。率(%)項目と金額項目でフォーマットを分岐。
*   **モーダル**:
    *   **ポートフォリオ追加モーダル**: 対象ポートフォリオの選択(セレクトボックス)、目標株価(任意)、メモ(任意)を入力し保存する。

### 2.3 分析手法画面 (AnalysisMethods)
*   **パス**: `/analysis`
*   **役割**: ユーザー独自のスクリーニング条件 (分析手法) を作成・管理し、実行結果を確認する。
*   **主要要素**:
    *   **新規手法の作成フォーム** (左カラム):
        *   手法名、説明
        *   **スクリーニング条件 (AND条件)**: `指標 (セレクト)`, `比較演算子 (>=, <=, == など)`, `数値` の動的リスト
        *   条件追加・削除ボタン、保存ボタン
    *   **保存済み手法リスト** (右カラム上部):
        *   手法名、説明、設定されている条件バッジのリスト
        *   「実行 (Screen)」ボタン、「削除」ボタン
    *   **スクリーニング結果** (右カラム下部 / 実行時のみ表示):
        *   表形式: `証券コード`, `企業名`, `最新書類の発行日`, `合致した指標の実績値`

#### 利用可能な指標一覧 (Available Metrics)
分析手法の作成において、以下の15種類の財務指標をスクリーニング条件として利用できます。

| 指標名 (日本語) | 識別子 (ID) | 単位 | 説明 |
| :--- | :--- | :--- | :--- |
| **売上高** | `net_sales` | 円 | 企業の主たる営業活動から得られた収益 |
| **営業利益** | `operating_income` | 円 | 売上高から売上原価および販管費を差し引いた利益 |
| **経常利益** | `ordinary_income` | 円 | 営業利益に営業外収益・費用を加減した利益 |
| **当期純利益** | `net_income` | 円 | 税金等を差し引いた最終的な利益 |
| **総資産** | `total_assets` | 円 | 企業が保有するすべての資産の合計 |
| **純資産** | `net_assets` | 円 | 総資産から負債を差し引いた自己資本 |
| **営業CF** | `operating_cf` | 円 | 本業による現金収支 |
| **投資CF** | `investing_cf` | 円 | 設備投資や資産売却による現金収支 |
| **財務CF** | `financing_cf` | 円 | 資金調達や返済による現金収支 |
| **1株当たり純利益 (EPS)** | `eps` | 円 | 当期純利益を、発行済株式数で割った値 |
| **1株当たり純資産 (BPS)** | `bps` | 円 | 純資産を、発行済株式数で割った値 |
| **ROE (自己資本利益率)** | `roe` | % | 自己資本を用いてどれだけ効率的に利益を上げたか |
| **ROA (総資産利益率)** | `roa` | % | 総資産を用いてどれだけ効率的に利益を上げたか |
| **自己資本比率** | `equity_ratio` | % | 総資産に対する純資産（自己資本）の割合 |
| **営業利益率** | `operating_margin` | % | 売上高に対する営業利益の割合 |

*   **モーダル**:
    *   **削除確認モーダル**: 手法の削除実行前の確認。

### 2.4 ポートフォリオ一覧画面 (Portfolios)
*   **パス**: `/portfolios`
*   **役割**: ユーザーが作成したポートフォリオのグループを管理する。
*   **主要要素**:
    *   タイトル、「新しいポートフォリオ」作成トグルボタン
    *   **新規作成フォーム** (展開時): ポートフォリオ名、説明入力フィールド
    *   **ポートフォリオカード一覧**:
        *   表示名、説明、登録銘柄数
        *   「削除」アイコンボタン (モーダルをトリガー)
        *   「詳細を表示して銘柄を管理」リンク (`/portfolios/:id` へ遷移)
*   **モーダル**:
    *   **削除確認モーダル (DeleteModal)**: ポートフォリオ全体の削除確認。

### 2.5 ポートフォリオ詳細画面 (PortfolioDetail)
*   **パス**: `/portfolios/:id`
*   **役割**: 選択したポートフォリオに含まれる具体的な銘柄(Portfolio Items)を一覧表示・管理する。
*   **主要要素**:
    *   「ポートフォリオ一覧へ戻る」リンク
    *   **ヘッダー**: ポートフォリオ名、説明
    *   **登録銘柄一覧表**:
        *   カラム: `証券コード / 企業名`, `業種`, `目標株価`, `メモ`, `追加日`, `操作 (解除ボタン)`
        *   企業名からは銘柄詳細画面 (`/stocks/:code`) へのリンク
*   **モーダル**:
    *   **解除確認モーダル (DeleteModal)**: 特定の銘柄をポートフォリオから外す前の確認。

### 2.6 認証関連画面 (Login / Register)
*   **パス**: `/login`, `/register`
*   **役割**: ユーザーの新規登録とログイン。
*   **主要要素**:
    *   フォーム: ユーザー名 (Registerのみ), メールアドレス, パスワード
    *   送信ボタン、エラーメッセージ表示、相互の画面への遷移リンク
